# minigame-1
## Devlog
Components are the instructions to build and place the Game Objects furniture in the Scene that is your house.
The Components control the Game Objects which is why they are the instructions. 
The Game Objects are the things in the Scene which is why they are the furniture. 
The Scene is your house because it is where the Game Objects are placed according to the size and coordinates in the Components.
